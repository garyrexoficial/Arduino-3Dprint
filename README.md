# Arduino & 3D Printing Project

This repository contains open-source projects that combine **Arduino programming** with **3D printing**.  
It includes code, technical documentation, and design files aimed at makers, students, and professionals interested in DIY electronics and rapid prototyping.  

### Features
- Arduino sketches for controlling electronic components.  
- 3D printed parts and accessories to integrate with hardware.  
- Protocols, guides, and comparative tables for troubleshooting and upgrades.  
- Licensed under **GNU GPL v3.0**, ensuring freedom to use, modify, and share.  

This project is part of my maker journey, blending **electronics, software, and design** to create practical solutions and educational resources for the community.


# Proyecto Arduino & Impresión 3D

Este repositorio reúne proyectos de código abierto que combinan **programación en Arduino** con **impresión 3D**.  
Incluye código, documentación técnica y archivos de diseño orientados a makers, estudiantes y profesionales interesados en electrónica DIY y prototipado rápido.  

### Características
- Sketches de Arduino para controlar componentes electrónicos.  
- Piezas y accesorios impresos en 3D para integrar con hardware.  
- Protocolos, guías y tablas comparativas para diagnóstico y mejoras.  
- Licenciado bajo **GNU GPL v3.0**, garantizando libertad de uso, modificación y distribución.  

Este proyecto forma parte de mi camino maker, integrando **electrónica, software y diseño** para crear soluciones prácticas y recursos educativos para la comunidad.
